CoinMarketCap JSON API

A straightforward API to access market information for the top 100 cryptocurrencies gathered from coinmarketcap.com. Updated every 5 minutes.

Open source, too!

Usage

Simply send a GET request to /api/symbol, where symbol is a currency code (e.g. BTC, LTC, DOGE, etc.)
You can also GET /api/all to receive information on all currencies at once.

Demos

  • /api/btc (Bitcoin)
  • /api/ltc (Litecoin)
  • /api/doge (Dogecoin)
  • /api/all (All supported currencies)

Property Isolation

If you want to only get a single property of a currency, you can add it to the URL after the currency code:

  • /api/btc/price (Bitcoin prices)
  • /api/ltc/market_cap (Litecoin market cap)
  • /api/doge/volume (Dogecoin volume over the last 24 hours)
  • /api/ppc/change (Peercoin price change over the last 24 hours)
